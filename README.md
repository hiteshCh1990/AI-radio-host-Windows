# Demo Files

This directory is intended for demo and example files.

## Example Script Format

Here's an example of a properly formatted radio show script:

```
Anjli: Hello dosto, welcome to Radio AI! Main hu Anjli.
Hitesh: Aur main hu Hitesh. Aaj Radio AI par hum baat karenge ek kaafi interesting topic ke baare mein.
Anjli: Ha yaar, toh bina time waste kiye chalo shuru karte hain aaj ka discussion.

Anjli: Toh dosto, aaj hum baat kar rahe hain Artificial Intelligence ke baare mein.
Hitesh: Haan bilkul! AI ab har jagah hai - phones mein, computers mein, aur ab toh radio mein bhi!
Anjli: [Laughs] Sach mein! Lekin AI kya hai exactly? Kuch log samajhte hain, kuch nahi.
Hitesh: Achha, toh simple terms mein, AI matlab hai machines jo soch sakti hain, seekh sakti hain.
Anjli: Exactly! Jaise hum insaan experience se seekhte hain, waise hi AI bhi data se seekhti hai.
Hitesh: Aur yeh kaafi fast hai - seconds mein kaam jo insaan ko hours lagte hain.
Anjli: Lekin ek sawaal - kya AI insaan ko replace kar degi?
Hitesh: Nahi yaar, AI toh tool hai. Insaan hi use karega, AI nahi.
Anjli: Bilkul sahi! AI se jobs create honge, kuch jobs change honge, lekin sab khatam nahi hoga.
Hitesh: Toh dosto, AI future hai, lekin insaan hi control karega.
Anjli: Ha yaar, technology insaan ke liye hai, insaan technology ke liye nahi.

Anjli: Toh dosto, umeed hai aaj ka discussion aapko pasand aaya hoga.
Hitesh: Agar pasand aaya ho toh Radio AI ke saath jude rahiye, aur aise hi interesting topics ke liye.
Anjli: Main hu Anjli,
Hitesh: Aur main hu Hitesh,
Anjli: Milte hain next episode mein, tab tak ke liye bye bye!
```

## Formatting Rules

1. **Speaker Prefix**: Each line must start with either `Anjli:` or `Hitesh:`
2. **Alternating Pattern**: Speakers should alternate (Anjli, Hitesh, Anjli, Hitesh...)
3. **Hinglish**: Use natural Hinglish (mix of Hindi and English)
4. **Casual Tone**: Keep it conversational and radio-friendly
5. **No Narration**: Only dialogue, no stage directions or narration

## Tips

- Keep dialogues natural and conversational
- Use casual fillers like "ha yaar", "achha", "bilkul"
- Add occasional laughter markers: [Laughs]
- Keep sentences relatively short for better audio flow
- Include questions and responses to make it interactive
